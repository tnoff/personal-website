Discord Bot

A discord bot framework written in python. Requires a valid discord bot token.

Includes basic database support, metadata commands, and plugin support via importing a base "Cog" class. This base cog class also includes support for retries of your custom commands.

You can view and download the source code on github.

I keep some discord plugins I maintain in a separate repo, so they are not included by default. There are some example plugins I've written available in github. These plugins include support for:

  • Music: Play audio from youtube in Voice Chat
  • Markov Bot: Mimic user chat using Markov chains
  • Urban Dictionary: Lookup words in urban dictionary
  • Role Management: Manage roles in hierarchical fashion
  • Auto Message Delete: Delete messages from channel on timer

Hathor Podcast Manager

Hathor is a python package and command line tool that can be used to download and manage media files from RSS Feeds, Souncloud Channels/Profiles, and Youtube Channels.

The package includes two command line scripts:

  • hathor: CLI Script for Hathor package
  • audio-tool: A small cli that can edit audio file metadata

Data from hathor is stored in a SQLITE datbase, where a "podcast" configuration will contain the following:

  • Name: Internal reference name given to the podcast
  • Arhive type: Either an RSS feed, Soundcloud Channel, or Youtube Channel
  • Broadcast ID: The identifier of the podcast feed.

Once hathor has this data for a given podcast, a sync can be run to gather the latest "episodes", and download them if requested. When a given episode is downloaded, hathor can also configure the audio metadata for the file.

In addition, plugins can be added to hathor, which can be configured to run after any function found in the main client.py code.

You can view and download the source code on github.

Backup Tool

Backup Tool is a python package and command line tool that can be used to encyrpt and backup local files to Oracle Cloud Infrastructures Object Storage. Once backed up to OCI Object Storage, files can also be downloaded and decrypted to restore local copies of files.

Backup Tool uses a basic AES encryption method with a password of 16, 24, or 32 characters in length. Files are encrypted before they are uploaded to OCI Object Storage to be backed up, and local md5 sums of the original file and the encrypted file are stored in a local SQLITE database.

Encrypted files can then be downloaded, and decrypted using the same password. Backup tool will verify the downloaded file and the decrypted files md5 sum against what was saved in the database.

You can view and download the source code on github.
